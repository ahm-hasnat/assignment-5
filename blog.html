Question-1: What are the different ways to select an element in the DOM?

There are various ways to select an element in the DOM.Among them getElementById(),getElementsByClassName(),getElementsByTagName(),querySelector(),querySelectorAll(),getElementsByName() these are most commonly used property.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

To set HTML content including tag name and element we use innerHTML.To set the text content of an element we use innerHTML. text.Content is used to set all text content including hidden text without HTML tag.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we use a single eventListener to a parent element instead of multiple eventListener to its children.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in DOM when an event is trigered it stars at the target element then bubble up to its ancestor.

Question-5: How do you create, add, and remove elements using JavaScript?

In js we can create, add and remove elements using various DOM methods.Such as to create an element we use "document.createElement()" Add an Element to the DOM-appendChild(),append(),to remove- removeChild() or directly use remove()
